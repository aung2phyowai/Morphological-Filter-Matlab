Fast Morphological Filtering Function For Matlab

Author: Michael Stevens

Date: Initial date of creation unknown. Date of upload to Github: December 16 2014 14:51 pm AEST.

This project contains 3 functions: fastMorphFilt, fastClose, fastOpen.  fastClose and fastOpen perform rapid morphological filtering on 1-D data using a flat structuring element.

fastMorphFilter perfoms opening and closing operations and performs a weighted combination of the output.
